家計レポートの更新方法（毎月）

このページは、毎月のレポート更新で「何を・どこに置くか」を思い出すためのメモです。

毎月やることは1つだけ：マネーフォワードの「資産推移」CSVを、決まったフォルダに入れる。
あとはPCで 「家計レポート更新して」 と頼めば、新しいレポートがこのメールで届きます。
1

データを用意する

マネーフォワードME から 「資産推移」の月次データ を CSV でダウンロードします。

これが毎月入れ替える唯一のデータです。

2

フォルダに入れる

ダウンロードした CSV を、エクスプローラーで次の「投入口」フォルダに入れるだけです。

投入口（ここに入れるだけ） I:\マイドライブ\20_claude code\00_マスタデータ置き場\

ファイル名はそのままでOK。中身を見て、自動で正しい場所（家計_資産データ）に振り分け＆リネームされます。フォルダの奥まで入れる必要はありません。

💡 家計のデータも仕事のデータも、まずはこの「00_マスタデータ置き場」に入れるだけ。あとは次のステップで頼むだけです。
3

更新をお願いする

PCで Claude Code を開いて、こう頼みます：

家計レポート更新して

数値の反映・PDF作成・メール送信まで、まとめてやってくれます。

4

メールで受け取る

新しいレポートが Gmailに自動で届きます（件名「【家計レポート】最新版」）。

スマホはその添付PDFを開くだけ。ドライブを探す必要はありません。

迷ったら、これだけ覚えればOK

CSVをフォルダに入れて

「家計レポート更新して」

NISA・ローン・保険など他の資料が届いたときも、同じフォルダに入れて同じ言葉で頼めば反映されます。

家計レポートは PC の Claude Code で作成・送信しています。
このページは毎月のメールからいつでも開けます。
